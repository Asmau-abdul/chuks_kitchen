# Chuks Kitchen — Frontend Documentation

> A responsive food ordering web interface built for Trueminds Innovations Ltd.
> **Client:** Mr. Chukwudi Okorie (Mr. Chuks) | **Business:** Chuks Kitchen

---

## Project Overview

Chuks Kitchen is a static, responsive frontend for a Food Ordering & Customer Management System. Customers can browse authentic Nigerian meals, add items to a cart, and place orders — all without a backend, APIs, or authentication.

This deliverable was built as part of the Trueminds Frontend Developer Internship (Feb 13–27, 2025).

---

## Tech Stack

| Tool / Technology | Purpose |
|---|---|
| **HTML5** (semantic) | Page structure and accessibility |
| **CSS3** | Layout, animations, responsiveness |
| **JavaScript** | Interactivity, cart logic, DOM manipulation |
| **Google Fonts** | Playfair Display (display), DM Sans (body) |
| **CSS Flexbox & Grid** | Responsive multi-column layouts |
| **CSS Media Queries** | Mobile, tablet, and desktop breakpoints |

No frameworks, no build tools, no dependencies — just clean native web.
